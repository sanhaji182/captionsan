# CaptionSan — Revision Loop Subtasks

## Subtask 6.1: Manual edit flow
- Allow users to edit output directly.
- Keep the last AI output and the edited output separate.
- Save the edited result.

## Subtask 6.2: Revision instruction form
- Add free-text revision input.
- Send revision instructions to AI.
- Support multiple revision rounds.

## Subtask 6.3: Revision history
- Save each revision event.
- Store instruction text and resulting content.
- Keep an audit trail of changes.

## Subtask 6.4: Approval flow
- Add approval status.
- Let users explicitly approve a final version.
- Prevent accidental auto-approval.
