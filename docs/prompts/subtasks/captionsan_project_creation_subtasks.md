# CaptionSan — Project Creation Subtasks

## Subtask 3.1: Project schema
- Define project model.
- Include title, original input, additional context, source language, and status.
- Preserve original user text exactly.

## Subtask 3.2: Create project UI
- Add a form for main idea and rough draft.
- Allow optional context input.
- Save the project as draft or ready state.

## Subtask 3.3: Project list/history
- Add a project list view.
- Show recent projects.
- Make it easy to reopen a previous brief.
