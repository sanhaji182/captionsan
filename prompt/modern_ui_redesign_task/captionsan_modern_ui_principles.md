# CaptionSan — Design Principles for Modern Redesign

Use these principles while redesigning CaptionSan.

## Visual direction
- Light mode first.
- Calm and professional, not flashy.
- Spacious layout with clear hierarchy.
- Warm but restrained color accents.
- Subtle shadows, soft borders, and rounded surfaces.

## Information hierarchy
- Show one primary action per screen.
- Keep advanced features behind progressive disclosure.
- Make status and next steps visible at a glance.
- Use section grouping to reduce cognitive load.

## Dashboard behavior
- Action-first home screen.
- Recent activity and pending review in prominent positions.
- Empty states should teach and guide.
- CTAs should be specific, not generic.

## Component style
- Consistent spacing and radius.
- Strong typography hierarchy.
- Clear button priority levels.
- Status badges that are readable and quiet.

## Motion and interaction
- Motion should clarify state changes.
- Avoid decorative animation.
- Keep transitions short and smooth.
- Use hover and focus states to reinforce affordance.

## Discovery and onboarding
- Use contextual hints instead of long tours.
- Make onboarding skippable.
- Show advanced tools only when relevant.
- Guide users with empty states and helper text.
