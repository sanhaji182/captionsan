# Task 7 — Accessibility and Validation

Validate the redesign for accessibility and consistency.

Requirements:
- Check contrast, keyboard flow, readable hierarchy, and responsive behavior.
- Ensure bilingual text still fits correctly.
- Verify core workflows remain intact.
- Cover major pages and states with tests or QA checks.

Acceptance criteria:
- The redesign is usable and readable.
- Core flows are not broken.
- The update is ready for review.
