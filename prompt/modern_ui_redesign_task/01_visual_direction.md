# Task 1 — Visual Direction

Define the modern visual direction for CaptionSan.

Requirements:
- Make the UI feel modern, calm, and professional.
- Use light mode as the default.
- Keep the layout spacious with clear hierarchy and strong contrast.
- Preserve prompt-first workflow, bilingual UI, and invite-only access.

Acceptance criteria:
- The design direction is explicit.
- The app feels lighter and more contemporary.
- No core workflow is changed yet.
