# Task 3 — Dashboard Redesign

Redesign the home dashboard to be action-first and status-aware.

Requirements:
- Show clear primary actions like create prompt, continue draft, and review pending.
- Highlight status, recent activity, and next action.
- Make the dashboard feel polished and modern.
- Keep Indonesian as the default UI language with English support.

Acceptance criteria:
- The dashboard has one obvious next action.
- Users can understand project state at a glance.
- The design feels clean and premium.
