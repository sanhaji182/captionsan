# Task 5 — Components and Visual System

Refine the component style and visual system for consistency.

Requirements:
- Define consistent spacing, radius, shadows, typography, and button hierarchy.
- Make cards, forms, tables, and status badges feel cohesive.
- Keep the system simple enough for future expansion.
- Avoid overly decorative or noisy styling.

Acceptance criteria:
- Core components look consistent.
- The interface feels cohesive across pages.
- The style is modern without becoming flashy.
