# Task 2 — Review and Approval Loop

Strengthen the prompt review and approval workflow.

Requirements:
- Let users review generated prompts before content generation.
- Allow edits and regenerate flow when the prompt is not approved.
- Track approved, revised, and pending states clearly.
- Keep version history compatible with the workflow.

Acceptance criteria:
- Users can approve or revise prompts safely.
- The workflow remains simple and predictable.
- No approved prompt is overwritten silently.
