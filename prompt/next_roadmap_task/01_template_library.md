# Task 1 — Prompt Template Library

Build a reusable Prompt Template Library for CaptionSan.

Requirements:
- Support reusable prompt templates for different content types.
- Allow templates to be organized by platform, tone, and use case.
- Keep Indonesian as the default UI language with English support intact.
- Preserve the prompt-first workflow.

Acceptance criteria:
- Users can browse, create, edit, and reuse prompt templates.
- Templates are modular and easy to extend.
- The feature fits the existing app structure.
