# Task 4 — Operational Dashboard

Build an operational dashboard for jobs, results, and workflow status.

Requirements:
- Show generation jobs, statuses, outputs, and recent activity.
- Support filtering by project, platform, status, and date.
- Keep the UI lightweight and practical for frequent use.
- Preserve the app’s prompt-first structure.

Acceptance criteria:
- Users can monitor job progress quickly.
- The dashboard is usable for day-to-day operations.
- The design stays aligned with the app style.
