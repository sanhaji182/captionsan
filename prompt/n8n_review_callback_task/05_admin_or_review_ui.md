# Task 5 — Admin or Review UI

Add a UI surface to inspect n8n review callback results.

Requirements:
- Show callback status, reviewer notes, timestamps, and related job data.
- Keep Indonesian as the default UI language with English support.
- Make it easy to see what happened and what action is needed next.
- Match the current app style.

Acceptance criteria:
- Users can inspect review callback results clearly.
- The UI is readable and consistent.
- Manual follow-up actions remain simple.
