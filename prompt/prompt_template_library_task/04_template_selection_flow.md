# Task 4 — Template Selection Flow

Connect the template library to the prompt generation flow.

Requirements:
- Allow users to start from a template and customize it.
- Keep the prompt-first workflow intact.
- Support reusing templates across different projects or content types.
- Make the selection flow simple and predictable.

Acceptance criteria:
- A template can be selected and turned into a working prompt.
- Users can edit before generation.
- Existing generation behavior is not broken.
