# Task 3 — Template CRUD API

Create backend APIs for managing prompt templates.

Requirements:
- Support create, read, update, delete, and list operations.
- Add filtering by platform, tone, use case, and status.
- Protect access using existing auth rules.
- Keep the API modular and easy to extend.

Acceptance criteria:
- Templates can be managed through the API.
- Filtering works correctly.
- Unauthorized access remains blocked.
