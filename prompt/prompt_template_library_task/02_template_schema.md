# Task 2 — Template Schema

Design or refine the schema for prompt templates.

Requirements:
- Support reusable templates with metadata for platform, tone, use case, and language.
- Allow draft, published, and archived states if needed.
- Keep compatibility with PostgreSQL and Drizzle ORM.
- Preserve existing records safely.

Acceptance criteria:
- Templates can be stored and retrieved reliably.
- Schema supports future extension.
- Current data remains intact.
