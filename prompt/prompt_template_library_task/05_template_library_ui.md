# Task 5 — Template Library UI

Build the Prompt Template Library interface.

Requirements:
- Show template lists, filters, detail views, and edit forms.
- Keep Indonesian as the default UI language with English support.
- Match the app’s existing style and accessibility patterns.
- Make browsing and reusing templates easy.

Acceptance criteria:
- Users can manage templates from the UI.
- Filtering and selection are clear.
- The UI remains consistent and responsive.
